# Mazingira Cloud 🌍

**Regenerating the Congolese land/fields through climate and soil data intelligence.**

Mazingira Cloud is an AI-powered environmental data platform for Africa, starting with the Democratic Republic of the Congo. It combines open-source environmental intelligence with premium private analytics for agriculture and climate resilience.

## 🌐 Platform Overview

### Open Data Layer (Free)
- Real-time interactive satellite map of the Congo Basin
- NDVI, rainfall, temperature, and soil moisture data
- Downloadable datasets (CSV & API access)
- Climate alerts and monitoring

### Premium Intelligence (SaaS)
- AI-powered soil and crop analysis
- Yield forecasting and climate risk scoring
- Personalized regenerative agriculture recommendations
- Custom PDF reports

### AI Assistant
- Natural language environmental queries
- Smart crop and irrigation recommendations
- Predictive climate modeling

## 🛠 Tech Stack

- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS
- **UI**: shadcn/ui components, Framer Motion animations
- **Maps**: Leaflet with satellite imagery tiles
- **Charts**: Recharts for data visualization
- **Backend**: Supabase (auth, database, edge functions)
- **AI**: Mazingira AI assistant (powered by Lovable AI Gateway)
